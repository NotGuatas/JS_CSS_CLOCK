# JS + CSS Clock  
Reto del curso **JavaScript30** implementado con HTML, CSS y JavaScript.  
Este proyecto replica un reloj analógico cuyo funcionamiento se basa en la rotación dinámica de las manecillas utilizando la hora actual del sistema.  

Además, añadí una **mejora extra**: un **reloj digital** que muestra la hora en formato `Hora:Minuto:Segundo`.

---

## Reto seleccionado
**02 - JS + CSS Clock – JavaScript30 (Wes Bos)**

---

## Objetivo del reto
Construir un reloj analógico utilizando únicamente JavaScript, calculando la posición de cada manecilla mediante transformaciones CSS basadas en la hora actual.  
El reto ayuda a comprender:
- Manipulación del DOM.
- CSS transform.
- Uso del objeto `Date`.
- Actualización en tiempo real con `setInterval`.

---

## Mejoras implementadas
- Un reloj digital sincronizado debajo del reloj analógico.  
- Mantiene la estética y estructura original del reto pero con una forma más fácil de visualizar que hora es realmente.  
- Ajuste mínimo del layout para colocar ambos relojes de forma alineada.
